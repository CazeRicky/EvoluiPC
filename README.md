# EvoluiPC
EvoluiPC: Planejamento inteligente de upgrades de hardware. Através de Grafos de Conhecimento, mapeamos gargalos e compatibilidades para gerar uma 'árvore de evolução' personalizada. Maximize sua performance com o melhor custo-benefício, evite erros técnicos e reduza o lixo eletrônico com uma rota de hardware precisa e orientada por dados.
