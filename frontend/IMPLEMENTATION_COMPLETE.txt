## ⚡ CONFIGURAÇÃO RÁPIDA

1. Obter IP:      ipconfig
2. Editar .env:   VITE_API_BASE_URL=http://SEU_IP:8000/api
3. Iniciar Django: python manage.py runserver 0.0.0.0:8000
4. Iniciar Frontend: npm run dev
5. Abrir: http://localhost:5173
